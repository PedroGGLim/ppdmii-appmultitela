## 📌 Descrição do Projeto

Este projeto consiste no desenvolvimento de um aplicativo Android em Java que funciona como um **Hub de Utilidades**, reunindo múltiplas funcionalidades em um único sistema.

O objetivo principal é centralizar diferentes aplicações desenvolvidas anteriormente (MegaSena, IMC e Jokenpô) e integrá-las com um novo módulo chamado **Questionário da Felicidade**, permitindo a navegação entre telas (Activities) de forma simples e intuitiva.

---

## 🧩 Funcionalidades do Hub

O aplicativo possui um menu principal que dá acesso às seguintes ferramentas:

* 🎲 **MegaSena**: Geração de números aleatórios simulando um jogo da loteria.
* ⚖️ **IMC**: Cálculo do Índice de Massa Corporal com classificação.
* 🪨📄✂️ **Jokenpô**: Jogo interativo contra o computador com sistema de pontuação.
* 😊 **Questionário da Felicidade**: Avaliação baseada em sono e nível de estresse, com cálculo ponderado e classificação final.

---

## 🔄 Navegação

O app utiliza múltiplas **Activities** e navegação via **Intents**, permitindo:

* Acesso às funcionalidades a partir do menu principal
* Transição entre telas
* Retorno ao menu através de botão "Voltar" ou navegação padrão do Android
